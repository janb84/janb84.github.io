GUIX on NixOS

Setting up NixOS for GUIX

16 MAY 2025 | 5 MIN READ - 30 MIN WORK

terminal in cyber

Trustless software development and reproducible builds are at the heart of Bitcoin's ethos, and Guix is the key to making this a reality. By enabling bootstrappable, auditable builds, Guix empowers developers to verify every step of the Bitcoin Core compilation process, minimizing reliance on precompiled binaries.

Setting up a reproducible Bitcoin Core build environment using Guix on a Mac can be a hassle due to its dependency on a Linux base. In this blog, I'll show you a simple solution: running NixOS on UTM to create a clean, isolated environment for compiling Bitcoin Core with Guix.

NixOS install

Once the VM is correctly set up, start the VM and boot from the NixOS installer.

After booting, you will be presented with the NixOS installer screen.

NixOS installer screen

Select the first option to start the installation process. NixOS will boot into a live environment. You might see a warning about "no display" - this is normal when running in UTM. Just wait.

UTM warning screen

Once booted, you'll be logged in as root in a command-line interface, where you can configure and install NixOS.
